    collection.push(e1, e2, e3..);
    collection.splice(startIndex, deleteCount, e1, e2, e3);

          the existing elements.

                                                                   Object Type

- It is used to keep all related data and logic together.
- Alan Kay introduced the concept of object into computer programming in early 1960's.
- Object enables separation and reusability.
- Object is a set of keys and values.

Syntax:
        {
           key : value,
           key : value
        }

- Key is always string type.
- Value can be any type.
    a) Primitive
    b) Non Primitive
    c) Function
 
Syntax:
      var product = {
          id : 1,
          name: "TV",
          price: 45000,
          cities: ["Hyd", "Delhi"],
          rating: { rate: 4.2, count: 5000 },
          qty: 2,
          total: function(){
              return qty * price;
          }
      }

- You can access the keys within object by using "this" keyword and outside object by using object name.

        total : function() {
             return this.qty * this.price;
         }

        product.total();
        product.price
        product.name
        product.rating.rate

- You can assign new values into object

        product.price = 56000;
        product.rating.rate = 4.9;

- If object comprises of a representation of only data then it is referred as "JSON".
  [JavaScript Object Notation]

- JSON is light weight and faster in interactions.
- It doesn't require COM to marshal [Common Object Model]
  COM is converting the Object code to Binary and vice versa.
